## Spy Countries
*Version 1.0.0*

A simple beginner app with Ionic 3

### Requirements

- npm 3.10.10
- nodejs 6.10.3
- cordova 7.0.0
- ionic 3.2.0

### Tips

Remember to run ``npm install`` after ```git clone```
To test on a real device ```ionic cordova platform add android``` or ```ionic cordova platform add ios```
